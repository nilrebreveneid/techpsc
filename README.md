# TECHPSC

This repository is set up to manage TECHPSC work through GitHub.

It is designed to support:

- contributor onboarding
- task assignment
- work submissions through pull requests
- contributor points tracking
- finance requests and approvals
- weekly updates
- reviewer ownership

## Repository workflow

Use GitHub in this way:

- Issues for tasks, finance records, and weekly updates
- Pull requests for submissions and review
- Project board for points, status, approvals, and finance tracking
- Labels for category and priority

## Included management files

- `.github/ISSUE_TEMPLATE/` for structured issue forms
- `.github/pull_request_template.md` for submissions
- `CONTRIBUTING.md` for contributor workflow
- `docs/project-setup.md` for project board fields and views
- `docs/migrate-current-sheets.md` for spreadsheet migration guidance

## Next steps

1. Upload the contributor points and finance sheets.
2. Create the GitHub Project called `TECHPSC Master Board`.
3. Add the custom fields described in `docs/project-setup.md`.
4. Start opening issues for all tasks, records, and updates.
